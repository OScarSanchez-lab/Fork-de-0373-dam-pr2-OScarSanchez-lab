## 📝 Descripción
Un panel de control interactivo para la visualización de cada destino. La interfaz permite al usuario acceder a la información de manera dinámica, atractiva e interactiva.

## 🌐 Contexto
Este dashboard fue desarrollado para el módulo de **Lenguajes de Marcas** (Web) en **La Salle Gràcia**. El reto consistió en crear una página web uzando todo lo aprendido en clases y usando los lenguajes definidos, además se implemto APIs que permitieron realizar acciones como la temperatua a tiempo real o la conversion de monedas.
